How it works

  1. Upload two files of any supported type.
  2. We extract text (incl. OCR for images).
  3. We vectorize both texts using CountVectorizer.
  4. Cosine similarity is computed → percentage score.
  5. Result is rendered instantly on this page.

Tips for accurate results

  • Prefer text-rich PDFs over scanned images when possible.
  • Ensure images are clear for better OCR.
  • Avoid uploading empty or password-protected files.

FAQ

Why did I see random values before?
Earlier demo HTML used a simulated client-side percentage. This page now relies solely on the server’s /check route, so the number reflects the true cosine similarity.
Does it detect paraphrasing?
Bag-of-words is sensitive to wording. For paraphrase robustness you could add n-grams, TF-IDF, or embedding-based similarity later.
Are my files stored?
They are processed in-memory for this check and not persisted by default.
